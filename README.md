A Vue.js SFC components library

## How to test components

    On Windows: `vue serve .\src\lib-dev.vue`
    On Linux: `vue serve ./src/lib-dev.vue`

## How to build and publish the library

    # update the version number
    npm run build
    npm login
    npm publish
